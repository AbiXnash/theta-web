# Animations and UX

## Animations

- Section reveal animations on Home/Events/Contact
- Hover lift and border/glow responses on cards/buttons
- Sponsor lane marquee animation with manual interaction pause

## Skeletons

Events route renders skeleton placeholders when content is not ready.

## Keyboard and Interaction QoL

- `Escape` closes modals
- `/` focuses Events search input
- Scroll-to-top floating action in app shell

## Accessibility

- `focus-visible` styles on interactive elements
- Reduced motion support for users with motion sensitivity
