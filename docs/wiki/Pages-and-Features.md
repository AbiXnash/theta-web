# Pages and Features

## Home (`/`)

- Hero with event branding + countdown card
- Festival day cards with modal schedule view
- About section
- Animated stats counters
- Sponsor lanes (tiered, animated, modal on click)
- CTA block

## Events (`/events`)

### Explorer mode

- Search
- Filters: category, status, cluster
- Event cards with badges and fee/status
- Event detail modal

### Coming-soon mode

Driven by `content.json -> eventsPage.comingSoon.enabled`.

- Replaces explorer UI with branded maintenance screen
- CTA and highlights are JSON-driven

## Contact (`/contact`)

- Team sections rendered by configured order
- Team cards with avatar fallback (`/team/default-avatar.svg`)
- WebTek support block
- Still-have-questions CTA block
